# S3 Storage — Plan d'implémentation

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Implémenter le sous-projet "S3 Storage" du backend NestJS (`StorageModule` `@Global` exposant un `StorageService` générique, container MinIO local, ~9 tests, ~108 tests au total) selon le spec `2026-04-25-s3-storage-design.md`.

**Architecture:** Module NestJS d'infrastructure pure, calque exact `DatabaseModule`. Token `S3_CLIENT` (Symbol) injecté via factory créant un `S3Client` depuis `AppConfigService`. `StorageService` exporté avec 5 méthodes génériques (`upload/get/delete/list/getPublicUrl`). Pas de controller HTTP, pas d'interaction DB. `OnModuleDestroy` pour cleanup HTTP keep-alive.

**Tech Stack:** NestJS 11, `@aws-sdk/client-s3` v3 (avec `forcePathStyle: true` pour MinIO/Garage), MinIO container (S3-compatible, dev local), `aws-sdk-client-mock` + `@smithy/util-stream` (tests), Podman compose.

**Référence spec :** `docs/superpowers/specs/2026-04-25-s3-storage-design.md`

- [ ] **Step 3: Implémenter `src/storage/storage.service.ts`**

```typescript
import { Inject, Injectable, NotFoundException } from '@nestjs/common';
import {
  S3Client,
  PutObjectCommand,
  GetObjectCommand,
  DeleteObjectCommand,
  ListObjectsV2Command,
  NoSuchKey,
} from '@aws-sdk/client-s3';
import { AppConfigService } from '../config/app-config.service';
import { S3_CLIENT } from './s3.constants';
import type { S3Object } from './storage.types';

@Injectable()
export class StorageService {
  constructor(
    @Inject(S3_CLIENT) private readonly s3: S3Client,
    private readonly cfg: AppConfigService,
  ) {}

  async upload(
    bucket: string,
    key: string,
    body: Buffer,
    contentType: string,
  ): Promise<void> {
    await this.s3.send(
      new PutObjectCommand({
        Bucket: bucket,
        Key: key,
        Body: body,
        ContentType: contentType,
      }),
    );
  }

  async get(bucket: string, key: string): Promise<Buffer> {
    try {
      const res = await this.s3.send(
        new GetObjectCommand({ Bucket: bucket, Key: key }),
      );
      if (!res.Body) {
        throw new NotFoundException(
          `S3 object ${bucket}/${key} has empty body`,
        );
      }
      return Buffer.from(await res.Body.transformToByteArray());
    } catch (err: unknown) {
      if (err instanceof NoSuchKey) {
        throw new NotFoundException(`S3 object ${bucket}/${key} not found`);
      }
      throw err;
    }
  }

  async delete(bucket: string, key: string): Promise<void> {
    // S3 DeleteObject est idempotent : pas d'erreur si la clé n'existe pas.
    await this.s3.send(
      new DeleteObjectCommand({ Bucket: bucket, Key: key }),
    );
  }

  async list(bucket: string, prefix?: string): Promise<S3Object[]> {
    const res = await this.s3.send(
      new ListObjectsV2Command({ Bucket: bucket, Prefix: prefix }),
    );
    return (res.Contents ?? []).map((o) => ({
      key: o.Key!,
      size: o.Size ?? 0,
      lastModified: o.LastModified ?? new Date(0),
    }));
  }

  getPublicUrl(bucket: string, key: string): string {
    const base = this.cfg.s3PublicUrl.replace(/\/$/, '');
    return `${base}/${bucket}/${encodeURIComponent(key)}`;
  }
}
```

  inject: [AppConfigService],
